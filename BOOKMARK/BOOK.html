Social Media Content Analyzer & Organizer

Project Overview

Develop an innovative application that:

  • Collects share links from YouTube, Instagram, Facebook, and Telegram
  • Analyzes collected content using state-of-the-art, free, and fast AI models
  • Organizes analyzed content into categorized bookmarks
  • Enhances content outlines with additional relevant information
  • Integrates Gemini AI models for advanced conversational insights

Project Goals

  • Improve content discovery and organization for users
  • Provide in-depth content analysis for better understanding
  • Offer a platform for discussion and insight generation using AI

Gemini AI Model Integration

Utilize Gemini's Conversational AI for generating summaries or discussions around the analyzed content.

Integration: API Key from Gemini, integrate using backend (Node.js) to process and display enhanced content insights.

Processing Time (Approx.): 2s - 5s

Output Formats: Text Summary, Conversational Dialogue

Bookmark Organization Structure

  • Tier 1: Platform (YouTube, Instagram, Facebook, Telegram)
  • Tier 2: Content Type (Video, Image, Text)
  • Tier 3: Category (Automatically tagged using spaCy/NLTK)
  • Tier 4: Content Outline with Summary (Generated using integrated AI models)
  • Tier 5: User Notes and Tags (Optional for personalized organization)
